About the project

What Should I Watch Tonight? is a Flash web application that recommends films based on a short interactive quiz. It combines front-end quiz interaction with server-side recommendation logic in Python.

How it works

The user answers five multiple-choice questions about mood, pace, focus, runtime, and the genre.

Each film in the application is stored in Python as structured data. When the user submits the quiz, the application compares their answers with the stored film attributes and returns the closest match.

The site also includes a random film option for users who want to skip the quiz and let the application make the choice for them.

In addition, users can submit film suggestions through a form. These suggestions are processed in Python and displayed back on the page, demonstrating form handling, POST request processing, and dynamic content rendering with Flask.
